INTRODUCTION

To assist breeders in making parental selections and tracking breeding materials within their breeding programs, we have developed maxPedigree©, a program for managing pedigree and trait data for cultivars and breeding lines. maxPedigree© Vesrsion 1.0 is developed using JavaScript,HTML and CSS. The maxPedigree© interface reads data from CSV file and is able to pull the pedigree and trait information for visualization and application by plant breeders.
The current version of maxPedigree© features:

  1. An interactive search, that will allow breeders to look up and visualize pedigree tree information.
  2. Phenotypic data for descriptors and agronomic traits established by the USDA for the soybean germplasm collection.
  3. Exportation of customizable data tables with desired lines and trait data.
  4. Different color for parent gender (Blue for Male and Pink for Female)

Future features will include:
  1. Colored trait tracing through pedigree trees.
  2. Storing of genotypic data for haplotype calling and population structure/diversity analyses.
  3. Integration of established tools for marker-assisted selection (MAS), genome-wide association study (GWAS) and genomic selection (GS) applications. Each breeding program can have their own personal database for storing cultivar and breeding line information with maxPedigree©.

READ ME

Uploading data :

  1. Download the template to upload your cultivar data in predefined format. Columns headings of the template file must not be changed.
  2. Cultivar_ID and Cultivar_Name columns values must be provided in datasheet.
  3. Always save your datasheet in .csv (comma sepeated values) format. for example, data.csv.
  4. Template File contains test data when downloaded. User can keep that data or delete it. DO NOT CHANGE TEMPLATE FILE HEADER ROW.

Export Filtered data and search bar :

  1. The search bar can be used to filter( based on Cultivar,Maturity Group,Year,Origin) the uploaded data.
  2. The Filtered data can be exported into new csv file by using Export Filter Data button.

Pedigree visualization :

  1. Show pedigree button will show the pedigree structure. tree can be zoomed and dragged.
  2. click on any cultivar name (or the Red Colored box) will display its information in Cultivar info box.
  3. Any one attribute can be selected to be displayed in Tree using attribute selection box.

Limitations of Version 1.0 :

  1. MaxPedigree version can Display first 1000 data rows of datasheet. However, the Search, Filteration and show Pedigree function works on whole datasheet.
  2. Pedigree tree works great until five generations. However, you might see overlapping of nodes in display after that.
  3. Use of collapsing and expanding function (CLICK ON Cultivar CIRCLE NODE) can help overcome the overlapping of nodes.
